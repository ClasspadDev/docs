Install HollyHock2

In order to get the custom programs to run on your classpad, you'll have to "mod" it using a custom patched rom. In plain english, that mean you'll modify the software that run on your calculator.

Simple install​

First, you'll need the newest release of HollyHock2, grab it here : https://github.com/SnailMath/hollyhock-2/releases. Scroll down to the "Snail2021.zip" and follow the instruction :

  • Extract the folder Snail2021 to the Desktop
  • Run Snail2021.exe
  • Follow the instructions on screen (You need these things: http://angusj.com/resourcehacker/ and https://tiplanet.org/forum/archives_voir.php?id=1044960 )
  • When you installed the update on the calculator, disconnect and reconnect it and copy run.bin, Tetris30.hhk and snake.hhk onto the calculator.
